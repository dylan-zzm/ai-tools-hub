# AI工具导航平台

一个帮助用户发现和使用各类AI工具的导航网站。

## 功能特点

- 分类展示各类AI工具
- 工具卡片展示
- 最近访问记录
- 节日倒计时提醒
- 响应式设计

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## 本地开发

```bash
# 克隆项目
git clone [你的仓库地址]

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
